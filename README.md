## Built with 

### Series	

          ZOHOIC CHIP Z1

### Clock Rate

          2110 - 3780 MHz

### Number of Cores / Threads	

          6 / 6 OR 8/8

### Transistor Count

          19000 Million OR 20000 Million

### Manufacturing Technology

           3 nm

### Features

          16-Core Neural Engine / 18-Core Neutral Engine 


### GPU	

          ZOHOIC GPU G1
